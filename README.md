# Konsultera Solutions Private Limited

This repository contains the web application for Konsultera Solutions Private Limited (also registered as Konsultera Solutions Pvt. Ltd.). As an organisation, Konsultera serves as an extended technology team for enterprise clients. The firm delivers premium technology consulting and custom software engineering solutions in the domains of Artificial Intelligence (AI), Machine Learning (ML), Cloud Services, DevOps, Big Data Analytics, Internet of Things (IoT), and Open Source Consulting.

The web application is designed to be highly responsive, optimised for search engines, and visually engaging.

---

## Core Technology Stack

The application leverages modern technologies for scalability and performance:

- **Framework**: TanStack Start (incorporating TanStack React Router and Vite for full-stack React capabilities).
- **Library**: React 19 (utilising TypeScript).
- **Styling**: Tailwind CSS version 4.
- **Tooling**: Vite (as the primary build tool and development server).

## Getting Started

Kindly follow the instructions detailed below to run the project on your local system.

### Prerequisites

Please ensure that you have Node.js and npm installed on your machine. We recommend using the active Long Term Support (LTS) version of Node.js.

### Installation

To install all the required dependencies, please run the following command in your terminal:

```bash
npm install
```

### Running the Development Server

To launch the local development server, execute the following command:

```bash
npm run dev
```

The application will be accessible at `http://localhost:8080`.

### Building for Production

To generate an optimised production build, run the command below:

```bash
npm run build
```

### Previewing the Production Build

You may preview the built application locally using the command below:

```bash
npm run preview
```

### Linting and Code Formatting

In order to maintain quality and consistency, run the commands below:

- To run the linter and detect issues:
  ```bash
  npm run lint
  ```
- To format the files across the project:
  ```bash
  npm run format
  ```
